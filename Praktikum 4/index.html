Tentang Saya

Hai, Saya Indira Ayu Anggraeni seorang mahasiswa semester 4 jurusan Informatika di Universitas Singaperbangsa Karawang. Saat ini saya sedang belajar Pemrograman Berbasis Web dan sangat ingin mendalami HTML, CSS, JavaScript.

Project 1

Project ini merupakan salah satu tugas yang ada di mata kuliah PBW yang dimana saya membuat biodata mengenai diri saya sendiri. Menggunakan HTML dan CSS yang sangat cocok untuk pemula. Dan saya memuat beberapa informasi penting disana dan contact person agar bagi siapa saja yang ingin bekerja sama memudahkan mereka untuk mengaksesnya.

Project 2

Salah satu project PBW saya yang sangat mudah untuk dibuat, karena terlihat simple dan mudah dimengerti berisikan tabel data penjualan untuk salah satu toko buku yang ada di Karawang, dan mereka bisa mengaksesnya dan sangat mudah untuk dijangkau oleh siapapun. Ini menggunakan HTML dan CSS dasar.
